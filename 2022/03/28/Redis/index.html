Redis是一款key-value存储结构的内存级NoSQL数据库

windows版本

在其安装目录中打开命令行启动服务端：
redis-server.exe redis.windows.conf

在新的命令窗口启动客户端：redis-cli 开始进行数据库操作

set name value set name zzw

get name zzw

hset key field value hset keya a1 zzw

hget key field hget keya a1 zzw
